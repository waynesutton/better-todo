Usage Examples

CSS Custom Properties

Use these CSS custom properties in your stylesheets:

/* Background colors */
background-color: var(--cursor-bg-editor);
background-color: var(--cursor-bg-sidebar);

/* Text colors */
color: var(--cursor-text-foreground);
color: var(--cursor-text-keywords);

/* Syntax highlighting */
.keyword { color: var(--cursor-syntax-keyword); }
.string { color: var(--cursor-syntax-string); }
.comment { color: var(--cursor-syntax-comment); }

JavaScript/TypeScript

Access colors programmatically:

// Get computed style values
const root = document.documentElement;
const editorBg = getComputedStyle(root)
  .getPropertyValue('--cursor-bg-editor');

// Set colors dynamically
root.style.setProperty('--cursor-bg-editor', '#1a1a1a');

VS Code Theme Integration

These colors match VS Code's "Dark+" theme used by Cursor:

// In your VS Code theme JSON
{
  "colors": {
    "editor.background": "#1e1e1e",
    "editor.foreground": "#d4d4d4",
    "editor.selectionBackground": "#264f78"
  },
  "tokenColors": [
    {
      "scope": ["keyword"],
      "settings": { "foreground": "#569cd6" }
    },
    {
      "scope": ["string"],
      "settings": { "foreground": "#ce9178" }
    }
  ]
}
